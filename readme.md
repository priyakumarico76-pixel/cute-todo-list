# Cute Sticky Note To-Do List

A small aesthetic sticky-note style to-do list app built using Python and Tkinter.

## Features
- Add tasks
- Cute pink sticky note design
- Simple GUI
- Beginner friendly project
- add custom backgrounds
- delete selected tasks

## Built With
Python  
Tkinter

## How to Run

1. Install Python
2. Run the file

python todolist.py
